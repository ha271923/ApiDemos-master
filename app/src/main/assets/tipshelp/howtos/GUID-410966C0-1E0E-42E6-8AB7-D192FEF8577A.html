Checking battery history

Check how long you've been using [generic phone name] since the last charge. You can also see a chart that shows how long the screen has been on, and how long you've been using connections like the mobile network or Wi-Fi.
  1. Slide the Notifications panel open, and then tap .
  2. Tap Power > History.
  3. If you've used [generic phone name] for a long time since the last charge, swipe left or right to scale the graph that details battery usage over time.

    You can also spread your fingers or pinch to scale the graph.
